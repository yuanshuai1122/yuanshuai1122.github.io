3、案列

先订阅后发布 后才能收到消息，

1 可以一次性订阅多个，SUBSCRIBE c1 c2 c3

2 消息发布，PUBLISH c2 hello-redis

3 订阅多个，通配符*， PSUBSCRIBE new*

4 收取消息， PUBLISH new1 redis2015
